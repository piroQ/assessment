HTMLについて

Web ページ（ホームページ）を作るための言語です。
Hyper Text Markup Language（ハイパーテキスト・マークアップ・ランゲージ）の略で、そのまま「エイチティーエムエル」と読みます。

様々なHTMLタグ

HTMLタグは、文章をマークアップさせることで、文章に意味を与えることができます。

pタグは、文章を表すので、pタグごとに隙間が空いた改行になります。

div このタグは他のタグとは異なり、何の意味も持ちません。
ただし、JavaScript のプログラムや CSS から利用したい時には、
この div タグを使ってマークアップしておくと、
そこにプログラムから何かしらの表示をさせたり、
スタイルシートで表示を変更したりすることができて便利です。

CSSを使ったプログラミング

CSS は、Cascading Style Sheets （カスケーディング・スタイル・シート） の略称で、スタイルシートと呼ばれることもあります。
